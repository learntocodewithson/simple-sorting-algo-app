# Simple Sorting Algo App

# Structure
 - OOP
# Variables
 - data, choice
# Database
 - none
# Program Flow
 - Display title
 - Display menu
  - Options:
   - 1.) Bubble Sort
   - 2.) Selection Sort
   - 3.) Insertion Sort
   - 4.) Merge Sort
   - 5.) Quick Sort
   - 6.) Heap Sort
   - 7.) Redix Sort
 - Input: 
  - Enter a sequence of numbers <separated by space>: 
   - getline, split string by space and store it in array
 - Display Result
  - UNSORTED 
  - ASCENDING ORDER
  - DESCENDING ORDER
